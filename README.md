# Portfolio - Vedansh Gupta

Personal portfolio website showcasing skills, projects, and contact information.

## Deployment

To deploy this project on Vercel:

1.  Push the code to GitHub (already done).
2.  Log in to [Vercel](https://vercel.com/).
3.  Click **"Add New..."** -> **"Project"**.
4.  Import this repository (`Portfolio-Vedansh`).
5.  Click **"Deploy"**.

Vercel will automatically detect the static site and deploy it.
